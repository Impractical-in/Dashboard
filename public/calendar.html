Calendar Shortcuts
Drag: move task to another day/time slot.
Resize bottom edge: increase/decrease task duration.
Alt/Ctrl + Drop: copy task to target day/time.
Double-click: open origin task in To-Do.
Shift + Double-click: span task across multiple days (linked series).
Right-click: delete one occurrence or entire linked series.
Linked series: moving/resizing one occurrence updates all in series.
